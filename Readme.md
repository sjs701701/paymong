this is paymong remake project base on https://paymong.kr/

## Design Asset Paths

### Header logo

Place the header SVG here:

- `public/brand/paymong-header-logo.svg`

### Hero keyword assets

Place keyword-specific hero assets under:

- `public/design/hero-keywords/rent/`
- `public/design/hero-keywords/tuition/`
- `public/design/hero-keywords/labor/`
- `public/design/hero-keywords/contract/`

Each keyword folder should contain these files:

- `orbit-shell.webp`
- `right-ribbon.webp`
- `left-sweep.webp`
- `bottom-echo.webp`
